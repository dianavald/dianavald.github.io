Abstract Mexico

The main objective of this project was to develop innovative products, inspired by some elements of the Aztec’s Mexican roots.

The designs started as an abstract representation of regular shapes ‘grecas’ founded on the paths on stones, paintings on walls. And the result were functional objects (made of oak wood) with an inspirational background.
